一些 CT 图像中常见伪影的矫正或缓解的**解析**方案

### 伪影包括：
- 杯状伪影 (Cupping artifact)
  - 水束矫正 (ECC)
  - 散射矫正 (SKS, ASKS)


- 硬化伪影 (Beam harden artifact)
  - 骨硬化矫正 (EBHC)
  - 金属硬化矫正 (TODO)


- 环状伪影 (Ring artifacts)


- 截断伪影 (Truncated artifact)
- 有限角伪影 (Limited view artifact )


- 其他 (正在学习补充……)



### 每个伪影文件夹下主要包括：
- 伪影的表现形式 (图片)
- 伪影的产生原因
- 经典矫正方案
- 代码实现

【注】该项目也是 [crip](https://github.com/z0gSh1u/crip) 工具包的**伪影矫正**使用范例，结合食用效果更佳！
